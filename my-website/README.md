# Physical AI & Humanoid Robotics Book

This website was created with [Docusaurus](https://docusaurus.io/) and deployed on [Vercel](https://vercel.com).

## Installation

```bash
npm install
```

## Local Development

```bash
npm run start
```

This command starts a local development server and opens up a browser window. Most changes are reflected live without having to restart the server.

## Build

```bash
npm run build
```

This command generates static content into the `build` directory and can be served using any static contents hosting service.

## Deployment

This project is configured for deployment on Vercel. To deploy:

1. Push your code to a GitHub repository
2. Connect your repository to Vercel via the [Vercel Dashboard](https://vercel.com/dashboard)
3. Vercel will automatically detect this is a Docusaurus project and build/deploy it

Alternatively, you can deploy directly using the Vercel CLI:

```bash
npm i -g vercel
vercel
```
